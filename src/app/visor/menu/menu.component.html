    Controles

    Menú de grupos y componentes

    • Debes seleccionar primero un grupo para poder después seleccionar un componente.
    • El ícono en los componentes indica que éste tiene un marcador asociado. Al hacerle clic, hay que escanearlo con la cámara para poder activarlo. Si el componente no tiene el ícono, con solo hacerle clic se desplegará el componente.
    • Una vez cargado el componente, aparecerá el ícono Éste puede contener información adicional sobre el contenido que se despliega. Si es un modelo, también muestra una imagen del marcador asociado.

    Acciones disponibles

    (Únicamente para dispositivos móviles)
    • Cambiar el tamaño del objeto
    • Mover el objeto
    • Rotar el objeto
